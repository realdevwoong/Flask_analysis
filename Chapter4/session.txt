세션(session) 과 쿠키(cookie) 는 웹에서 사용자 상태를 유지하기 위한 핵심 기술
웹은 stateless (상태 없음) -> 요청(Request) 하나가 끝나면 서버는 사용자가 누군지 기억x
쿠키와 세션을 통해 사용자를 구분하고 데이터를 유지

쿠키 (Cookie)

브라우저(클라이언트) 쪽에 저장되는 작은 데이터 조각 (key-value)

세션 (Session)

서버 쪽에서 사용자 정보를 저장하고 관리

사용자를 구분하기 위해 세션 ID 라는 고유값을 발급

브라우저는 세션 ID를 가지고 요청 → 서버가 “어떤 사용자”인지 식별
